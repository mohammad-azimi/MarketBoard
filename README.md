Django Marketplace

This is a simple project for creating a classified ads website where users can sign up, post ads, and sell their items. It is similar to platforms like Craigslist or Gumtree, allowing users to list products and services for sale. This project is implemented using Django.

Project Status 🚧

The project is not yet complete and is under development. I wrote this project during my workout time and used YouTube videos and various educational resources to learn and implement it.

Main Features ✅

User registration and login

Ability to create, edit, and delete ads

Display ads on the homepage

Search and filter ads (under development)

Installation and Running 🚀

Clone the repository:

git clone https://github.com/USERNAME/REPOSITORY_NAME.git
cd REPOSITORY_NAME

Create and activate a virtual environment:

python -m venv env
source env/bin/activate  # For Windows: env\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Apply database migrations:

python manage.py migrate

Run the local server:

python manage.py runserver

Now, you can visit http://127.0.0.1:8000/ to see the project in action.
